# Gemini IA - XD

## Descrição
Este projeto é uma forma de acessar o Gemini IA em redes com restrições. Ele não utiliza o link oficial do Gemini, mas direciona para uma versão que funciona em determinados ambientes escolares.

## Como Funciona
O funcionamento é simples:

1. Um link específico foi configurado para levar a uma versão web do Gemini IA.
2. Ao acessar o link, o usuário é redirecionado para a versão funcional, sem precisar de permissões especiais.
3. Nenhum software adicional é necessário, apenas acesso ao link via navegador.
